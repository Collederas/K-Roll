# Contributing to Kroll

Thanks for your interest in contributing.

Kroll is a single-maintainer project, but contributions are welcome and appreciated.

## How to contribute

You can contribute by:
- Reporting bugs or unexpected behavior
- Suggesting improvements or features
- Submitting pull requests with fixes or enhancements

Please keep pull requests focused and reasonably scoped.

## License

## Communication

If you are unsure whether a change makes sense or want early feedback, feel free to open an issue before starting work.
